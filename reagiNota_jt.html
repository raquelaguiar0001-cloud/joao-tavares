⚠️ Como sincronizar:
1. No dispositivo A → clique Baixar Backup
2. Transfira o arquivo para o dispositivo B (e-mail, WhatsApp, Drive…)
3. No dispositivo B → clique Carregar Backup
4. Os dados são mesclados — nada é apagado
